Instruction Guide

Launch Requirements

Before launching the IDE, ensure your environment is ready for autonomous agent execution.

1

Install Codex CLI

The IDE requires the codex binary to be available in your PowerShell/Terminal path. This drives the AI logic.

2

Launch executable

Double-click Codex-IDE-Final.exe. A console window will appear showing the backend status.

3

Automatic Start

Your default browser will automatically open http://localhost:8000. If it doesn't, navigate there manually.

⚠️ Administrative Rights:

The IDE modifies files in its directory. Run as Administrator if you encounter file permission errors.
